# RubyDebugger

简单的ruby调试器

突然感觉调试器也很简单的

于是自己弄一个简单的看看。。

也就是断点，给一个irb

## Installation

Add this line to your application's Gemfile:

    gem 'ruby_debugger'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby_debugger
